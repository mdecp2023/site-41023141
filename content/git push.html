git push

1.如果你要將更新推送至遠端,首先必須要有使用者名稱和信箱

 你可以在命令列上輸入:

  git config --global user.email "you@example.com"

  git config --global user.name "Your Name"

如果輸入成功你應該會在home_ipv4或6(看你用什麼開)裡找到叫.gitconfig的檔案

你的使用者名稱和信箱就存在裡面

2.你必須要有token或者ssh(我在其他頁面有講)

3.最後你可以用acp " *** "指令推送( *** 是你本次版本更新的名稱(內容))
